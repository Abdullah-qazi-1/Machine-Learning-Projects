# End-to-End Machine Learning Project

A complete ML project built from scratch as part of my Data Science learning journey. Used **student.csv** dataset to predict student exam performance.

---

## What's Covered

- Data Ingestion
- Data Transformation using Pipelines
- Model Training & Evaluation
- Hyperparameter Tuning
- Prediction Pipeline
- Logging & Exception Handling

---

## How to Run

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
pip install -r requirements.txt
python src/components/data_ingestion.py
```

---

## Tech Stack

Python, Scikit-learn, Pandas, NumPy, XGBoost, CatBoost
